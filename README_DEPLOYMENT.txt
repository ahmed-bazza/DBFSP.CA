DBFSP.ca Static Website Package

Files:
- index.html: main homepage
- refer/index.html: referral form page
- assets/forms/referral-form-fillable.pdf: fillable referral PDF
- assets/forms/referral-form-fax.pdf: fax-friendly referral PDF
- robots.txt and sitemap.xml: basic SEO files

Deployment:
1. Upload all contents of this folder to the web root/public_html of dbfsp.ca.
2. Confirm the homepage loads at https://dbfsp.ca/.
3. Confirm referral page loads at https://dbfsp.ca/refer/.
4. Confirm PDFs load from https://dbfsp.ca/assets/forms/.
5. Set up referrals@dbfsp.ca with your domain email provider.
6. Confirm privacy/security process before accepting patient-identifiable referrals by email.
7. Update contact details if clinic phone/fax/address changes.

Suggested domain email:
- referrals@dbfsp.ca
- info@dbfsp.ca

No server-side code is included; this is a static site.
